# AI Data Chat frontend

Aplicacion standalone de Angular completada hasta Sprint 2. Incluye identidad, panel administrativo
y configuracion de conexiones/modelos con estados y capacidades. No almacena JWT, claves,
ciphertext ni credenciales en `localStorage` o `sessionStorage`.

La guia completa de ejecucion y alcance vive en el `README.md` de la raiz del monorepo.
