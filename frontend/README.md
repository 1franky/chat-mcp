# AI Data Chat frontend

Aplicacion standalone de Angular para el Sprint 1. Incluye bootstrap del primer administrador,
registro, login/logout con sesion server-side, guards de identidad y panel administrativo de
usuarios. No almacena JWT, secretos ni credenciales en `localStorage` o `sessionStorage`.

La guia completa de ejecucion y alcance vive en el `README.md` de la raiz del monorepo.
