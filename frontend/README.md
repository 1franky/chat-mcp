# AI Data Chat frontend

Aplicacion standalone de Angular completada hasta Sprint 3. Incluye identidad, panel administrativo,
proveedores/modelos y chat persistente con SSE, cancelacion, Markdown seguro y temas claro/oscuro.
No almacena JWT, claves, ciphertext ni credenciales en `localStorage` o `sessionStorage`; sólo la
preferencia visual de tema se conserva localmente.

La guia completa de ejecucion y alcance vive en el `README.md` de la raiz del monorepo.
